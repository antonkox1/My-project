Справочник CSS от Антона

С чего начать?

Обнуление стилей

*{
height: 100%;
padding: 0;
margin: 0;
border: 0;
}

Обращение к тегу по id (уникальный) с помощью # (применяется для логики, а не как свойство CSS) - #heading

Псевдоэлементы и псевдоклассы

Псевдоклассы:

a:visited - изменение после перехода

a:hover - при наведении мыши

a:active - при клике мышью

a:focus - при нажатии Tab

Псевдоэлементы:

p::first-letter - изменение первой буквы

p::first-line - изменение первой строчки

p::selection - изменение выделяемого цвета

h1::before{content: "Привет"} - добавляет текст сзади

h1::after{content: "Пока"} - добавляет текст спереди

.list(класс) li:ntn-clild(2) - изменяет определенное значение (можно использовать even и odd)

Блочная модель

display:block - блочный элемент, занимет всю ширину страницы

display:inline - строчный элемент, занимет необходимоую ширину, нет отступов снизу и сверху

display:inline-block - блочно-строчный элемент, встраевается как блок, но занимает определенное место

box-sizing: border-box (ужимает блок до опреденной высоты и ширины)

Позиционирование и размеры

position:relative - перещается относительно блока в котором находится

position:absolute - позицианируется относительно блока

position:fixed - позицианируется относительно окна браузера

position:sticky - при скороле смешается по блоку

Единицы, переполнение и уровни элементов

2em - увеличение шрифта в раза относительно родителя

2rem - увеличение шрифта в раза относительно базового шрифта в теге html

width: 100vw - занять 100 процентов ширины

height: calc(90vw - 100px) - занять 90 процентов высоты минус 100 пиксейлей

z-index: 1 - позицианирование на странице, чем выше число, тем выше позиционирование

overflow: auto - устанавливает скрол, если он нужен

overflow: scroll - устанавливает скрол всегда

overflow: hidden - обрезает контент

Границы и тени (Borders & Shadows)

border: red 5px solid - Внутренние границы

outline: blue 5px solid; - Внешняя граница

text-shadow: 1px 2px 5px orange; - первое значение смещение в право, второе смещение вних, третие размытие

box-shadow: -3px -2px 11px green; - К блоку

box-shadow: -3px -2px 11px green, 1px 2px 5px orange, 1px 2px 5px black; - inset тень внутри, можно применять несколько теней
