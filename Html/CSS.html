Справочник CSS от Антона

С чего начать?

Обнуление стилей

*{
height: 100%;
padding: 0;
margin: 0;
border: 0;
}

Обращение к тегу по id (уникальный) с помощью # (применяется для логики, а не как свойство CSS) - #heading

Псевдоэлементы и псевдоклассы

Псевдоклассы:

a:visited - изменение после перехода

a:hover - при наведении мыши

a:active - при клике мышью

a:focus - при нажатии Tab

Псевдоэлементы:

p::first-letter - изменение первой буквы

p::first-line - изменение первой строчки

p::selection - изменение выделяемого цвета

h1::before{content: "Привет"} - добавляет текст сзади

h1::after{content: "Пока"} - добавляет текст спереди

.list(класс) li:ntn-clild(2) - изменяет определенное значение (можно использовать even и odd)

Блочная модель

display:block - блочный элемент, занимет всю ширину страницы

display:inline - строчный элемент, занимет необходимоую ширину, нет отступов снизу и сверху

display:inline-block - блочно-строчный элемент, встраевается как блок, но занимает определенное место

box-sizing: border-box (ужимает блок до опреденной высоты и ширины)

Позиционирование и размеры

position:relative - перещается относительно блока в котором находится

position:absolute - позицианируется относительно блока

position:fixed - позицианируется относительно окна браузера

position:sticky - при скороле смешается по блоку

Единицы, переполнение и уровни элементов

2em - увеличение шрифта в раза относительно родителя

2rem - увеличение шрифта в раза относительно базового шрифта в теге html

width: 100vw - занять 100 процентов ширины

height: calc(90vw - 100px) - занять 90 процентов высоты минус 100 пиксейлей

z-index: 1 - позицианирование на странице, чем выше число, тем выше позиционирование

overflow: auto - устанавливает скрол, если он нужен

overflow: scroll - устанавливает скрол всегда

overflow: hidden - обрезает контент

Границы и тени (Borders & Shadows)

border: red 5px solid - Внутренние границы

outline: blue 5px solid; - Внешняя граница

text-shadow: 1px 2px 5px orange; - первое значение смещение в право, второе смещение вних, третие размытие

box-shadow: -3px -2px 11px green; - К блоку

box-shadow: -3px -2px 11px green, 1px 2px 5px orange, 1px 2px 5px black; - inset тень внутри, можно применять несколько теней

Трансформации (Transform)

transform: rotate(180deg) - переворот на 180 градусов

transform: scale(1.2) - увеличивает на 1.2 (если написать 0.8 то уменьшится)

transform: scaleX(1.2) - увеличивает по оси X на 1.2

transform: scaleY(1.2) - увеличивает по оси Y на 1.2

transform: translate(50%, 50%) - смещение по горизонтали и вертикали соответственно

transform: skew(21deg,21deg) - наклоны по горизонтали и вертикали соответственно

transform: skew(21deg,21deg) rotate(180deg) scaleY(1.2)- Множественное преобразование

transform-origin: 0 0; (top) - изменение точки опоры

Background

background-size: 100% 100% - размер фона

background-image: url(); - добавляет изображение

background-repeat: no-repeat - не повторяет изображение

background-position: 50% 50% - позицианирование фона(сейчас по центру)

background-attachment: fixed - фиксирует объект

background: linear-gradient (90deg, black 10%, white 50%, blue 80%) - добавляет градиет в 90 градусов

background: radial-gradient (90deg, black 10%, white 50%, blue 80%) - добавляет градиет в по кругу

Фильтры

filter:blur(2px) - Размытие

filter:drop-shadow (5px 6px 4px rgb (0,0,0, 0.5)) - создает тень (также и для псевдоэлементов)

filter:grayscale(100%) - задает черно-белым

filter:brighness(150%) - задает яркость

filter: opacity(50%) - задает прозрачность

Плавные переходы

transition-duration: 0.5s, 2s(500ms) - время анимации, можно задавать для нескольких анимаций

transition-property: background-color, transform (all) - к каким свойствам применить анимацию

transition-delay: 0, 500ms - время задержки анимации

transition-timing-function: linear - время задержки анимации

transition: all 1s ease .5s; - краткая запись

Анимации

animation-name: myAnimation - вызвать анимацию

animation-iteration-count: infinite(3) - задает количество анимаций

animation: name duration timing-function delay iteration-count direction fill-mode; - краткая запись

animation-play-state: paused(running) - останавливает или возобновляет анимацию

animation-fill-mode: forwards; - оставляет анимацию без возврата

@keyframe myAnimation {
from {background-color:blue}
50% {background-color:yellow}
to {background-color:black}
}

Медиазапросы

Полезности

visibility: hidden; - Скрыть текст (оставляет отступы)

column-count: 3 - разделяет текст на 3 колонки

column-rule: 1px solid black - добавляет линию разделения колонок
