Основные теги

HTML (HyperText Markup Language) — язык для разметки гипертекста, он является набором правил, по которым браузер отличает заголовки от списков, таблицы от картинок и так далее. HTML появился в 1993 году и был призван стандартизировать правила для вывода текста внутри веб-страниц.

Тег <p> является параграфом и добавляет отступы снизу и сверху

Тег <script src="Путь к файлу(js/script.js)"> подключается в конце перед закрывающим тегом /body

Теги для запонения формы

Тег form

<form action="#" method="get">

Атрибуты тега form:

action - задает url-адрес отправки формы

method - задает метод отправки данных

action - абсолютный url и относительный (http://site.com и на сервер)

methot="get" - посылает данные в строку запроса (стоит по умолчанию)

methot="post" - связь с сервером

Тег input

<input type="text" placeholder="Напиши что-нибудь" name="", value="">

Атрибуты тега input:

type - задает тип поля

name - задает имя поля (должно быть уникальное, кроме radio)

Кодить
