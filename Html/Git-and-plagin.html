Введение в Git от Антона

Начало работы

В терминале внести git init - установка репозитория

git config --local(global) user.name "Anton" (определить свое имя)

git config --local user.email antonkox1999@gmail.com (укощание имени)

git status - узнать статус проекта

git add -A - добавить все файлы в репозиторий

git commit -a -m"first commit" - создание точки (после m комментаий)

git log - история всех commit'ов

Как выложить Git в github

git remote add origin https://github.com/antonkox1/My-ptoject.git - связывает с github проект

git branch -M main - Связывает с github проект

git push -u origin main - Запушить в github c веткой

git push - Теперь запушить в github

Полезности

cd My-project(путь к файлу) - войти в папку

cd .. - выйти из папки

git pull - загрузить проект с github

git clone https://github.com/antonkox1/My-project.git - Копирует проект
