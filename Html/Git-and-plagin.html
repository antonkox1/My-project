Введение в Git от Антона

Начало работы

В терминале внести git init - установка репозитория

git config --local(global) user.name "Anton" (определить свое имя)

git config --local user.email antonkox1999@gmail.com (укощание имени)

git status - узнать статус проекта

git add -A - добавить все файлы в репозиторий

git commit -a -m"first commit" - создание точки (после m комментаий)
